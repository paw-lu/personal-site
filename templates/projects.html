GROCERY

A recommendation system predicting the grocery items a user will buy

The site is hosted on Heroku, so it will take a minute to load.

At The Data Incubator, I analyzed over 3 million orders from Instacart—a grocery delivery service. Using this data I was able to make a recommendation system that would predict the groceries people will buy. The model compares a user’s order history to that of all other customers in the database using tf–idf. The model then predicts what a user will order based on the purchase history of customers most similar to the user.

In addition to predicting orders using my model, I was also able to produce some interesting insights into people’s purchase habits. For example, there are some clear correlations between the time of the day and the type of foods that are bought. Did you know that cake reaches it’s peak consumption at 2:00 am? Click on the picture up top to visit the GROCERY website, then click the insights tab to learn other interesting trends about how people buy their food.

Dress up

A Unicode converter and demo of best Python practices

Dress up's command-line interface can be used to quickly convert between unicode characters.

Dress up is a library written in Python that makes converting between different Unicode types easy. A user can go from "words" to "𝔴𝔬𝔯𝔡𝔰", "🆆🅾🆁🅳🆂", and "𝔀𝓸𝓻𝓭𝓼" using Dress up as either a command-line interface or a Python library.

Most importantly, Dress up serves as a demonstration of what I think are the current best practices in Python development. It features strong linting via Flake8 and select plugins, high test coverage through pytest, generated documentation via Sphinx, type hinting, and a robust CI/CD framework powered by Nox and GitHub Actions.

Check out the repository at https://github.com/pscosta5/dressup!
